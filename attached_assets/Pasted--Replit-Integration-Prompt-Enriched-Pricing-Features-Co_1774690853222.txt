Supprimer les espaces multiples
Supprimer les mots consécutifs dupliqués
Correction automatique de fautes

COLORE → COLOREE, GOST → GHOST, JL M → JLM,
ORIGNE/ORGINE → ORIGINE, BOOSTE/BOSTER → BOOSTER,
POWR → POWER, SPRIN → SPRING
Extraction de champs structurés
Champ	Logique d'extraction
Famille moto	Détecter : GHOST, JLM, MBK, CIAO, POWER, BOOSTER, SPRING
Modèle	Famille + version/cylindrée (ex: "JLM 110", "GHOST V7")
Cylindrée	Extraire les nombres 50/80/100/110/125/150/200/250, formater "110cc"
Variante	Détecter : V1, V2, V7, ORIGINE, SPORT, RACING, PREMIUM, EVO
Couleur	Détecter : BLEU, NOIR, ROUGE, GRIS, BLANC, VERT, JAUNE, ORANGE, etc.

Afficher 3 décimales, format français (virgule)
Quand l'utilisateur change la marge, tous les prix se mettent à jour instantanément
Mettre le prix de vente en couleur verte pour le distinguer visuellement
